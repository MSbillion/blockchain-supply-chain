How the AI works

Isolation Forest (from scratch, pure JS): Each shipment is described as a feature vector: [weight, value, transit_days, temperature, stops, value_per_kg, delay_ratio]. The algorithm builds an ensemble of random isolation trees. Each tree recursively partitions the feature space by picking random attributes and random split thresholds. Anomalies — points that are statistically rare or extreme — get isolated in fewer splits (shorter path length). The anomaly score is computed from the average path length across all trees, normalised to [0,1] where scores above 0.62 indicate anomalies.

A* Route Optimisation: A weighted supply-chain graph connects major logistics hubs. A* uses a combined cost function f(n) = g(n) + h(n) where g(n) is actual cost (distance + risk penalty + congestion) and h(n) is the admissible heuristic (Euclidean distance to goal). It finds the lowest-risk, lowest-cost route for each shipment's origin-destination pair.
